# Conception And Development of a Personalized Travel Planning Application
## |Project completed in January 2023 at HOTIX|

A comprehensive web application for personalized travel planning that enables users to create optimized itineraries based on their budget, available time, and destination preferences.

## ‚ú® Key Features
# Administration
* Secure authentication for administrators
* Destination management: add, modify, delete tourist locations
* User management: supervision, deactivation and account monitoring
  
# Travelers
* User registration and authentication
* Budget management: set total budget and per-destination limits
* Time planning: configure maximum wait time per location
* Attraction selection: personalized management of points of interest
* Itinerary validation: approve or modify system suggestions
* Optimized routing: automatic generation of the most efficient path using Dijkstra's algorithm
* Emergency alert system: direct contact with moderators
* Profile management: customize user settings

# Moderation 
* Moderator authentication
* Location tracking: security monitoring of travelers
